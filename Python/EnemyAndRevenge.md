# EnemyAndRevenge x Poetry

Mista Playa

## Info

Check version
`poetry --version`

Install
`pipx install poetry`

Remove
`pipx uninstall poetry`

## Use

Creaate new project from 0
`poetry new ...`

Start config file **pyproject.toml** for exit project
`poetry init`

Add dependence
`poetry add ...`

Remove dependence
`poetry remove ...`

Generate file
`poetry install`

Active env
`poetry run ./manage.py` or `poetry shell`

## Settings

View configs
`poetry config --list`

### Change parameter

`poetry config virtualenvs.in-project true`
